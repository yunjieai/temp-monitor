### 🎯 建模工作流快速鏈

#### 基礎建模流程
```
添加模型 (Shift + A)
→ 進入編輯 (Tab)
→ 選擇 (A/1/2/3)
→ 邊迴圈切割 (Ctrl + R)
→ 倒角 (Ctrl + B)
→ 擠出 (E)
→ 移動/旋轉/縮放 (G/R/S)
→ 合併/分離 (M/P)
```

#### UV 拆解標準流程
```
標記接縫 (Ctrl + E → Mark Seam)
→ 全選 (A)
→ 展開 (U → Unwrap)
→ 進入 UV 編輯器
→ 優化排列 (P)
→ 檢查拉伸 (V)
```

## ⭐ 新手必學 5 步驟

### 第 1 步：掌握基本導航
```
中鍵旋轉視圖 → Shift + 中鍵平移 → 滾輪縮放
```

### 第 2 步：學會基本變換
```
Tab (進編輯) → G (移動) → R (旋轉) → S (縮放)
```

### 第 3 步：認識選擇模式
```
1 (頂點) → 2 (邊) → 3 (面) → A (全選)
```

### 第 4 步：掌握關鍵建模工具
```
E (擠出) → Ctrl + R (切割) → Ctrl + B (倒角)
```

### 第 5 步：學會 UV 拆解
```
Ctrl + E (標記接縫) → U (展開) → P (打包)
```
